# WeChat-ordering-system
这款基于微信小程序的点餐系统，为餐厅和顾客带来了极大便利。顾客无需排队等待，打开小程序就能轻松浏览菜单，查看菜品详情、价格和图片。点餐时可自由选择菜品、数量及备注要求，还能一键下单。餐厅端能实时接收订单，高效处理。支持线上支付，结账快速。此外，还能查看订单状态，了解出餐进度。该系统既提升了顾客的就餐体验，又提高了餐厅的运营效率，减少人力成本，实现双赢。
